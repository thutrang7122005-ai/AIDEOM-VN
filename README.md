Hệ thống Mô hình Ra quyết định Kinh tế Việt Nam (AIDEOM-VN)
Chào mừng bạn đến với đồ án cuối kỳ môn học liên quan đến mô hình hóa và ra quyết định. Dự án này được xây dựng nhằm cung cấp các giải pháp tối ưu hóa và phân tích dữ liệu phục vụ các bài toán kinh tế cụ thể tại Việt Nam.

🚀 Giới thiệu dự án
Ứng dụng được xây dựng trên nền tảng Streamlit, cho phép người dùng tương tác trực tiếp với các mô hình toán học và thuật toán ra quyết định phức tạp thông qua giao diện web trực quan.

🛠 Công nghệ sử dụng
Dự án được triển khai dựa trên các thư viện mạnh mẽ trong Python, sử dụng Windows Powershell run as administrator, công cụ claude hỗ trợ viết code:

Giao diện: Streamlit

Toán học & Tối ưu hóa: PuLP, Pyomo, CVXPY, Pymoo (NSGA-II)

Xử lý dữ liệu: Pandas, NumPy, Scipy

Trực quan hóa: Plotly, Matplotlib

📑 Danh mục các bài toán
Hệ thống bao gồm 12 bài toán kinh tế/ra quyết định, được phân chia theo các chuyên đề:

Tối ưu hóa nguồn lực: Phân bổ ngân sách, lập kế hoạch sản xuất.

Đánh giá & Xếp hạng: Sử dụng TOPSIS, Entropy để đánh giá vùng miền.

Tối ưu hóa đa mục tiêu: Áp dụng thuật toán di truyền (NSGA-II).

Các mô hình kinh tế khác: (Bổ sung thêm tên các bài toán nếu bạn muốn cụ thể hơn).

Truy cập vào trang web ứng dụng.

Nhập các tham số đầu vào (nếu có) hoặc xem kết quả phân tích dữ liệu tự động.

Kết quả sẽ được hiển thị dưới dạng bảng biểu hoặc biểu đồ tương tác.
